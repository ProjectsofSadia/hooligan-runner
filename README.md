# Hooligan Runner 🎵

Plays **Ha Ha Ha Hooligan by BTS** every time your code runs successfully in VS Code.

## Setup

1. Add your trimmed `hooligan.mp3` to the `media/` folder
2. Install the extension
3. Press `Ctrl+Shift+H` after a successful run to celebrate!

## Features

- Plays audio on successful task completion
- Manual trigger with `Ctrl+Shift+H`
- Windows supported via PowerShell

## Requirements

- Windows (PowerShell built-in)
- Your own `hooligan.mp3` file
